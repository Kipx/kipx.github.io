Tournament Brackets

Each season tourney is a two-stage tournament:

  1. A Swiss bracket lasting exactly 4 rounds.
  2. An adaptive single-elimination top cut lasting 0-2 rounds (given a player cap of 64 players).

The Swiss bracket is seeded using results from the previous 6 season tourneys, including tourneys from the season prior. The formula is (MatchesWon + 8) / (MatchesPlayed + 16), with ties broken randomly.

Each round’s match pairings are generated using Challonge (brackets hosted in the rLOR Challonge community). After 4 rounds, the first-stage ranking is determined using the default Challonge method, which is Match Score followed by four tiebreakers in this order:

  1. Match Score (1 point for each match win and bye; 0.5 points for each match draw)
  2. Match Tiebreaker (match wins against players with the same Match Score)
  3. Median-Bucholz (the sum all all opponents’ Match Scores, with largest and smallest discarded)
  4. Game Differential (the difference between game wins and losses)
  5. Initial Seeding, low seed ranking higher

After 4 rounds of Swiss, the tournament will enter a top 4 cut of the highest ranking players from the Swiss bracket. Ties are broken using the above tiebreakers.
